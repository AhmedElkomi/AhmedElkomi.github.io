ESL Real-Time Translator

A real-time gesture recognition system that translates Egyptian Sign Language into Arabic text and speech using cutting-edge AI.

Overview

ESL Real-Time Translator is a powerful AI application that uses computer vision and deep learning to recognize Egyptian Sign Language (ESL) in real-time. Designed for seamless desktop use with future plans for mobile optimization, it provides instant Arabic text and speech translations.

Technologies Used

Python MediaPipe Holistic OpenCV Bi-directional LSTM ONNX Runtime PyQt6 Arabic Text-to-Speech Custom ESL Dataset

Features

  • Real-time hand and pose tracking with MediaPipe Holistic
  • Bi-LSTM model trained on custom ESL dataset (93%+ accuracy)
  • Arabic TTS for spoken output of recognized signs
  • Context-aware phrase merging (e.g., combining letters into names)
  • Subtitle-style display for high-confidence predictions
  • Desktop GUI using PyQt6
  • Optimized for fast inference with ONNX Runtime

Challenges Solved

- Ensured robust real-time performance by optimizing frame extraction and feature encoding
- Addressed ambiguity in gesture sequences with context-aware NLP techniques
- Built a clean and responsive GUI while keeping latency under 100ms
